Introducción.

Aqui aprenderas a hacer diferentes platillos y comida tan deliciosa en tu casa.
Hay diferentes categorias para que puedas aprender el platillo o comida que mas te agrade.
Esta página te abrira el apetito, al ver los deliciosos platillos que hay en la página.


Objetivos.
  • Nuestra tarea es que el usuario se sienta satisfecho con la informacion.
  • Tenemos como fin hacer que el usuario adquiera conocimientos de gastronomia.
  • Nosotros queremos hacer que estos conocimientos gastronomicos se extiendan a mas lugares.
